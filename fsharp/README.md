## Setting Up

Run the F# REPL with:

```bash
./run repl
```

Execute an F# script with:

```bash
./run execute 1-introduction-to-simulation-methods.fsx
```

It takes around five minutes to run the command for the first time. This is because a fresh Docker image is built.
